---

## 📝 작품 개요
**킥보드 마스터**는 킥보드에 최적화된 길 찾기와 운전 점수에 따른 개별적인 경로 안내를 제공해 주는 애플리케이션입니다!

---

## 🛠 작품 기능 및 구현
✅ **사용자 로그인 / 회원가입** 후 원하는 목적지 입력  
✅ **운전 점수 기반 경로 안내**: 안전한 경로 또는 최단 시간 경로 제공  
✅ **GPS 및 자이로센서를 활용한 실시간 운전 데이터 측정**  
✅ **운행 종료 후 위반 사항에 따른 마일리지 차감 및 보상 제공**
